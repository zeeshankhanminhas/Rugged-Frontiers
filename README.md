# Rugged Frontiers — Pakistan Wilderness & Culture (Static Website)

A premium, minimal front-end concept website built with HTML5, CSS3, and vanilla JavaScript for GitHub Pages deployment.

## Structure

- `index.html`
- `experience-guide.html`
- `apply.html`
- `experiences/`
- `css/style.css`
- `js/main.js`
- `assets/images`, `assets/icons`, `assets/logo`

## Run locally

Open `index.html` directly or serve with a simple static server.
